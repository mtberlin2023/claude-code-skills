# home — domain pack

> Home is where decisions accumulate. Most are individually small; the bill at the end of the month is the cumulative truth. This is the pack to reach for when the problem is *a series of small choices that nobody has audited as a whole* — the §10 subscription-audit register from the paper, or any cumulative-budget shape.

---

## Roles (who does what)

| Concept | Home term | Use when |
|---|---|---|
| Principal | Head of household | The one who has to actually live with the call |
| Expert (any tier) | A trade you call in (plumber, electrician, accountant) | One narrow job; doesn't run the house |
| Meta-expert | Interior architect / financial advisor | Sets layout/structure; doesn't live there |
| Adversarial reviewer | House survey / energy assessor / nosy neighbour | Outside eyes that catch what habit normalised |
| Subagent (Haiku/Sonnet) | Cleaner / babysitter / part-time help | Owns one chore window; doesn't decide |
| Domain expert | Specialist (lawyer for a will, surveyor for a survey) | Called for one specific question |
| Sub-expert pair | Two opinions on the same renovation | Deliberately differing views on the same call |

## Gear (the tools and objects)

| Concept | Home term | Use when |
|---|---|---|
| Persona file | Household role / chore chart | Who's responsible for what, written down |
| Calibration card | The household budget / standing orders list | The reference picture of how much should leave each month |
| Instinct file | "How this house runs" — routines | What happens automatically without anyone calling it |
| Notepad / reflex bank | Fridge note / shared family chat | Reminders that catch you mid-day |
| Working memory (post-/dream) | "Remember when X happened" stories | The vivid lessons the household holds in eyeline |
| Hook (Claude Code) | Direct debit / scheduled bill / smoke alarm | Fires automatically without anyone deciding each time |
| MCP server | The pantry / supplies cupboard / loft | Where stored things are pulled from; not refilled in real time |
| Knowledge pack | Family recipe book / household manual | The how-we-do-it for one class of task |
| Skill | A learned domestic skill (cooking, DIY, parenting) | Portable across rooms and life-stages |
| Brief | A specific household project (loft conversion, new baby) | What we're taking on, before we commit |
| Project | A household initiative | Has a start, a budget, an end |
| Decision queue | The "to-decide" pile on the kitchen counter | Open questions waiting for someone to walk past with attention |

## Rhythms (how time and pace work)

| Concept | Home term | Use when |
|---|---|---|
| AWE (token effort) | Time + cash + emotional energy | What this decision actually costs the household |
| Session | A bounded household stretch | One contained run of activity |
| Session-open | Family breakfast / morning standup | Last reads before everyone scatters |
| Session-close | Bedtime routine + lock-up | Tidy, settle the day, set the alarms |
| Calibration cycle | Monthly bill review / annual budget review | Step back and look at the cumulative shape |
| Coaching pass | One-on-one with a family member about one habit | Targeted feedback, not the family meeting |
| Drift | "When did our bills get this high?" | Cumulative cost grew while nobody was watching |
| Cross-triage | "Is this normal for your house?" with a friend | Outside reference point |

## Verbs (what people do)

| Concept | Home term | Use when |
|---|---|---|
| Recommend | Suggest at dinner | Put it in front of the household |
| Veto | "Not in this house" | Hard no, no further discussion |
| Defer | "Let's wait and see" | Don't commit; revisit when more is known |
| Approve | "Go for it" | Green light |
| Calibrate | Rebalance the budget / shift the chores | Adjust the running totals |
| Iterate | Try a new routine | Same problem, different approach |
| Build (new expert) | Take on a new commitment / subscription / habit | New recurring line in the household |
| Retire | Cancel / discontinue | Off the standing-order list |
| Adversarial pass | Sit-down review | "Let's actually look at the accounts" |
| Stance protocol | Affordable / nice-to-have / safety check | Three lenses every household decision gets checked against |

## Freeform mappings

- **The kitchen table** — where the real decisions get made. Not the spreadsheet, not the inbox.
- **Standing orders / direct debits** — recurring commitments that quietly compound. The §10 subscription audit lives here.
- **The bills drawer** — where the open questions pile up until someone has the energy.
- **A free trial that quietly converted** — the paradigmatic accumulated-without-noticing item.
- **Spring cleaning** — periodic audit; the household calibration cycle made visible.
- **The junk drawer** — accumulated low-stakes items nobody has time to sort. Maps onto every backlog.
- **The cupboard at the back of the kitchen** — what's still there but nobody opens.
- **Routines vs novelties** — recurring expenses vs one-offs; budgets fail when novelties become routines silently.
- **The household budget** — the cumulative truth; nothing in it lies on its own, but the sum often does.
- **"Who's doing the school run?"** — capacity assignment under load. Everyone's role brief, every morning.

## When this analogy doesn't fit

- **Single high-stakes architectural calls.** Use construction. Home assumes things will continue, not be poured.
- **Tight technical workflows.** Cooking is faster and more precise. Home is too soft.
- **Adversarial / contractual disputes.** Home imports a collaborative frame that will hide real opposition.
- **Acute-cost decisions where speed beats cumulative care.** Home is built for slow accumulation, not crisis response.
